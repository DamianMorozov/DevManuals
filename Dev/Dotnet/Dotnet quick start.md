# Dotnet quick start

## Quick start v6
```
dotnet new console -o DotNetDemo -f net6.0
cd DotNetDemo
dotnet run
dotnet .\bin\Debug\net6.0\DotNetDemo.dll
```

## Quick start v5 [Deprecated]
```
mkdir DotNetDemo
cd DotNetDemo
dotnet new console
dotnet restore
dotnet build -c Release
dotnet .\bin\Debug\net5.0\DotNetDemo.dll
```
