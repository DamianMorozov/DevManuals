## Run console as admin.
```
nvm list
nvm install latest
nvm install 16.4.1
nvm install 20.11.0
nvm use latest
nvm use 16.4.1
nvm use 20.11.0
nvm current
npm -v
npm install -g yarn
yarn -v
yarn set version latest
yarn set version 1.22.19
```
