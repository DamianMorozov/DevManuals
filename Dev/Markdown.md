## Basic Syntax

**Heading**
```markdown
  # H1
  ## H2
  ### H3
```
**Bold**
```markdown
**bold text**
```
**Italic**
```markdown
*italicized text*
```
**Blockquote**
```markdown
> blockquote
```
**Ordered list**
```markdown
1. First item
2. Second item
3. Third item
```
**Unordered list**
```markdown
- First item
- Second item
- Third item
```
**Code**
```markdown
  `code`
```
**Horizontal rule**
```markdown
---
```
**Link**
```markdown
[title](https://www.example.com)
```
**Image**
```markdown
![alt text](image.jpg)
```

## Extended Syntax

**Table**
```markdown
| Syntax | Description |
| ----------- | ----------- |
| Header | Title |
| Paragraph | Text |
```
**Fenced Code Block**
```markdown
  ```
  {
    "firstName": "John",
    "lastName": "Smith",
    "age": 25
  }
  ```
```
**Footnote**
```markdown
Here's a sentence with a footnote. [^1]
[^1]: This is the footnote.
```
**Heading ID**
```markdown
### My Great Heading {#custom-id}
```
**Definition list**
```markdown
term
: definition
```
**Strikethrough**
```markdown
~~The world is flat.~~
```
**Task list**
```markdown
- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
```
