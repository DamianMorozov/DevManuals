Battle.net Linux
http://portwine-linux.ru/battle-net-linux/
$ cd ~/Downloads/
$ sudo apt install zenity xterm
$ sh PortBattleNET-80
$ sudo apt install wine
$ cd /home/user/PortWINE/PortBattleNET
$ sudo cp Battle.net.desktop /usr/share/applications
